# 股票分析助手 - 使用说明

## 🌐 在线版本

由于服务器安全限制，以下是几种使用方式：

### 方式 1：本地直接打开（最简单）✅

1. 找到文件：`/home/admin/.openclaw/workspace/stock-analyzer/static/index.html`
2. 双击用浏览器打开，或拖到浏览器
3. 输入股票代码即可使用

**优点**：无需安装，直接用
**缺点**：需要联网（获取股票数据）

---

### 方式 2：本地服务器运行

```bash
# 安装 serve
pnpm add -g serve

# 启动服务器
cd /home/admin/.openclaw/workspace/stock-analyzer/static
serve -l 3001

# 访问 http://localhost:3001
```

---

### 方式 3：部署到 Vercel（公开访问）

```bash
# 安装 Vercel
pnpm add -g vercel

# 登录 Vercel
vercel login

# 部署
cd /home/admin/.openclaw/workspace/stock-analyzer/static
vercel --prod

# 会返回一个 https://xxx.vercel.app 的链接
```

---

### 方式 4：部署到 GitHub Pages

1. 创建 GitHub 仓库
2. 将 `static/` 目录内容推送到仓库
3. 在仓库设置中启用 GitHub Pages
4. 访问 `https://用户名.github.io/仓库名`

---

## 📱 功能说明

- 输入股票代码（如：300308、600519）
- 查看实时行情
- 获取智能分析（估值、趋势、评分）
- 查看风险提示和亮点
