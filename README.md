# 股票分析助手 📈

一个实时股票行情查询与智能分析网页应用。

## 访问链接

🌐 **公网访问**: https://stock-analyzer-1772724897.loca.lt

## 功能特点

- ✅ 实时股票行情查询
- ✅ 智能估值分析
- ✅ 综合评分系统
- ✅ 风险提示与亮点分析
- ✅ 操作建议生成
- ✅ 响应式设计，支持手机/电脑

## 使用方法

1. 打开网页
2. 输入股票代码（如：300308、600519）
3. 点击"分析"按钮
4. 查看实时行情和分析报告

## 技术栈

- **前端**: HTML5 + CSS3 + JavaScript
- **后端**: Node.js + Express
- **数据源**: 新浪财经 API

## 项目结构

```
stock-analyzer/
├── frontend/
│   └── index.html      # 前端页面
├── backend/
│   ├── server.js       # 后端服务器
│   └── package.json    # 依赖配置
└── README.md
```

## 本地运行

```bash
# 安装依赖
cd backend
npm install

# 启动服务器
node server.js

# 访问 http://localhost:3000
```
